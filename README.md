# Simple crud project created to explore some tools

## Order Service Stack

* Spring
* Jetbrains Exposed ORM
* PostgreSQL
* Docker
* Kotlin
* Gradle
* GraalVM
* RabbitMQ

## Email Service Stack

* NodeJS
* JavaScript
* Nodemailer
* RabbitMQ

Use __"docker-compose up --build"__ to run the application on port 8080
